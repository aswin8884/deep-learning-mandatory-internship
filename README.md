# Deep Learning Architectures from Scratch (PyTorch)

Implementations of MLP, CNN, RNN, GRU, and LSTM built from the 
ground up in PyTorch as part of a Deep Learning Research Internship 
at the Institute for Computer Science, Universität Koblenz 
(graded 1.0 / with distinction).

Every architecture is implemented and trained without high-level 
abstractions, to demonstrate understanding of the underlying mechanics.
